### Что такое git?

Git - это широко используемая распределённая система контроля версий с открытым исходным кодом. Она позволяет легко разрабатывать как небольшие, так и крупные проекты.

Над одним проектом могут одновременно работать несколько разработчиков, которые могут отслеживать внесенные и отраженные в коде изменения. Кроме того, если что-то пойдет не так, вы сможете без труда вернуть изменения к предыдущей версии.

### Полезные команды

Для меня

1. Сделали очередные изменения? 
А оно относится к предыдущему коммиту, не хочешь много коммитов

Добавьте изменение: 
> git add хз.txt

Добавляем сообщения:
> git commit --amend -m "это хз"

Еще лучше без изменений комментария к коммиту:
> git commit --amend --no-edit 👌

Это создаст коммит на месте старого с вашими дополнениями.

Как запушить это, если предыдущее уже было запушено:
> git push --force origin main
